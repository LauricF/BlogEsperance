Tips to produce graphics

2020-05-21

Nice and constructive graphics play an important role when it comes to conveying a message.

I personnaly use R with ggplot2, a graphic package. the main advantages of ggplot is that is come with a grammar, a series of rules which once learn, make it very easy to create new plots. Thus I recommend to learn ggplot when starting to use R. It is time consuming at first and the curve is steep but it is totally worth in the long term.

However the following rules to produce graphics efficiently can be applied to any langage.

Do not reinvent the wheel

Whatever you want to display, there is a high chance that somebody have already done it.

For simple graph it is often quicker to do them from scratch but for complicated one when there is not a clear idea about what I really want to do, it is usually time-efficient to do a quick research:

Here are my favorit websites to start:

  • Data to viz, this really well organised website help to find the best graph given the data you have. There caveat section is very useful to implement clear and accessible plot. Code is in R, Python, D3.js and flourish.

  • The r graph gallery is useful to identify quickly an efficient way to display your data with all the necessary code to produce it.

  • ggplot2 extensions, a large number of packages have been made to extend the capability of ggplot2, going through the display items can give you ideas and save you a lot of time.

Look for new inspiration

There are many interesting websites showing new ideas, it is worth to spend few minutes each day/week to look into what is new;

  • Flowingdata website An incredible website which recense interesting new graphs.

  • data is beautiful subreddit

Ask for feedbacks

Constructive feedbacks from supportive people can dramaticaly increase the value of your graph. Do not underestimate how much a friendly glaze can increase your work.

Choice of colours

Choice of colours is important and can transform the perception and the time that people are ready to give to a graphic. It is particularly important to choose the right colour as often default colour are all but wonderful. I try to use colour blind people palette when possible;

  • Friendly palette of colour for colour blind peope

  • If you are not suffering from a color vision deficiency it is very hard to imagine how it looks like to be colorblind. The Color BLIndness Simulator can close this gap for you.
