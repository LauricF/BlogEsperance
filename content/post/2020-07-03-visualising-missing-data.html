Exploring a new data set

LAFerrat

2020-07-03

Beyond the classic functions like summary(), or str() , there is a set of dedicated packages to explore a new dataset. In this post, I will describe three useful packages which give a good perspective on a new dataset. The first one is visdat which gives tools to visualise at one glance a new dataset.

Its major functions are vis_dat,vis_miss and ,vis_cor vis_dat takes litteraly “looking at the data”. This is useful to identify pattern and potential biais in the data.

library(visdat)
vis_dat(airquality)

vis_miss gives an insight about missing values

vis_miss(airquality)

vis_cor which give a quick look at the correlation between the variables

vis_cor(airquality,
        na_action = "complete.obs")

For larger dataset, with more variables and/or a large number of observation I prefer to use naniar which gives tools to visualise at one glance a large dataset and particularly focus on missing values. The function gg_miss_upset shows which variables are missing simultanously.

library(naniar)
## Warning: package 'naniar' was built under R version 3.6.2
gg_miss_upset(airquality,nsets = 10)

Finally the package DataExplorer is a must. With one function a detailed report is generated with a pletoriade of graphs.

library(DataExplorer)
DataExplorer::create_report(airquality)
